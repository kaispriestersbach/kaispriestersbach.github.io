# Kai Spriestersbach | Operator Profile

A personal website styled as an FPS/tactical game character sheet, showcasing my career, skills, achievements, and life goals in a gamified format.

**Live:** [spriestersbach.github.io](https://spriestersbach.github.io)

## About

This project visualizes my professional journey and personal milestones as if they were stats, loadouts, medals, and missions from games like Helldivers 2, PUBG, or CS2.

**Features:**
- Tactical HUD-style interface with scanline effects
- Attributes displayed as RPG stats (max 20)
- Skills as weapon loadout with rarity tiers (Legendary/Epic/Rare)
- Achievements as earned medals
- Active objectives with progress tracking
- Completed missions (career milestones)
- Intel briefing (backstory)

## Setup

1. Clone this repository
2. Add your avatar image as `avatar.jpg` in the root directory
3. Deploy to GitHub Pages or any static hosting

## Customization

Edit `index.html` to modify:
- **Attributes:** Adjust values in the "Attributes" panel (format: `X/20`)
- **Loadout:** Change skills and their levels in the "Loadout" panel
- **Medals:** Add/remove achievements in "Medals Earned"
- **Objectives:** Update goals in "Active Objectives"
- **Missions:** Edit career milestones in "Mission Log"
- **Intel:** Modify backstory in "Intel Briefing"

## Tech Stack

Pure HTML/CSS, no dependencies. Uses Google Fonts (Orbitron, Rajdhani, Share Tech Mono).
